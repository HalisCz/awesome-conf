Mod1 = Klávesa "Alt", Mod4 = klávesa "Win"

Mod4 + F1 - Zobrazí nápovědu
Mod4 + F2 ... F5 - Zobrazí různá systémová hlášení a logy

Mod1 + F1 - Spustí terminál (xterm)
Mod1 + F2 - Spustí gmrun (gmrun)
Mod1 + F3 - Spustí dmenu (dmenu)
Mod1 + F4 - Ukončí aktivní aplikaci
Mod1 + F5 - Spustí vyhledávání na internetu pomocí browseru (firefox)
Mod1 + F6 - Spustí SSH připojení (ssh)
Mod1 + F7 - Spustí Rdesktop připojení (rdesktop)

Mod4 + Print - Provede screenshot obrazovky a otevře náhled (scrot, mirage)
Mod4 + c - Zamkne počítač (xscreensaver-command -lock)

Mod1 + PageUp - zesílí zvuk (pvol.py -p -c 5)
Mod1 + PageUp - zeslabí zvuk (pvol.py -p -c -5)
Mod1 + End - vypne/zapne zvuk (pvol.py -m)

Mod4 + s - Umístí/zruší plovoucí okno na všechny plochy
Mod4 + a - Nastaví/zruší možnost umístit plovoucí okno vždy nad ostatníma

Mod4 + Mod1 + Left - Zvětší šířku master okna o 5%
Mod4 + Mod1 + Right - Zmenší šířku master okna o 5%
Mod4 + Mod1 + Up - Zmenší výšku master okna o 5%
Mod4 + Mod1 + Down - Zmenší výšku master okna o 5%

Mod4 + q - Zobrazí/schová titlebar

Základní klávesové zkratky: # man awesome

ps.: Aplikace svázané s klávesovou zkratkou jsou v závorkách.
Nastavení klávesových zkratek je v souboru awesome.lua
