⊞ + F1 - Zobrazí nápovědu
⊞ + F2 - Zobrazí volné místo
⊞ + F3 - Zobrazí dmesg
⊞ + F4 - Zobrazí netstat
⊞ + F5 - Zobrazí procesy

Alt + F1 - Spustí terminál 
Alt + F2 - Spustí gmrun 
Alt + F3 - Spustí dmenu 
Alt + F4 - Ukončí aktivní aplikaci
Alt + F5 - Spustí vyhledávání na internetu pomocí browseru 
Alt + F6 - Spustí SSH připojení 
Alt + F7 - Spustí Rdesktop připojení 

⊞ + Print - Provede screenshot obrazovky
⊞ + e - Zamkne počítač

⊞ + s - Umístí/zruší okno na všechny plochy
⊞ + a - Nastaví/zruší plovoucí okno vždy nad ostatníma

⊞ + Alt + Left - Zvětší šířku master okna o 5%
⊞ + Alt + Right - Zmenší šířku master okna o 5%
⊞ + Alt + Up - Zmenší výšku master okna o 5%
⊞ + Alt + Down - Zmenší výšku master okna o 5%

⊞ + q - Zobrazí/schová titlebar


⊞ + Control + j	- Focus na další obrazovku
⊞ + Control + k	- Focus na předchozí obrazovku
⊞ + o			- Přesun okna na další obrazovku
